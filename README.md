JRaiDownloader
==============

Un downloader per scaricare i video dal sito della RAI.
Il programma supporta la sezione di Rai Replay (tutti i video/programmi andati in onda nell'ultima settimana).

Per scaricare un video, basta incollare nella textfield URL il link del video, e il programma lo scaricherà nella
cartella impostata (si può cambiare tramite il menu Impostazioni->Personalizza).
